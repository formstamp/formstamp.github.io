Date/Time pickers

A set of directives for entering date, time or both.

Directive fsDate

This directive creates a datepicker widget with dropdown calendar. This directive expects that scope property specified in ngModel will be a Date, and will modify only year/month/day components of this Date, time will remain unchanged. Supported attributes:

  • ng-model - see AngularJS documentation for ngModel directive
  • ng-disabled - disable/enable input with scope property
  • class - additional CSS classes 
Seppuku date: {{ selectedDate.toString() }}

Directive fsTime

This directive creates a time-input widget. This directive expects that scope property specified in ngModel will be a Date, and will modify only hours/minutes/seconds components of this Date, year/month/day will remain unchanged. Supported attributes:

  • ng-model - see AngularJS documentation for ngModel directive
  • ng-disabled - disable/enable input with scope property
  • class - additional CSS classes 
Time is: {{ svalue | json }}

Directive fsDatetime

Widget combining both fsTime and fsDate. Allows to enter both Date components.

  • ng-model - see AngularJS documentation for ngModel directive
  • ng-disabled - disable/enable input with scope property
