Datepicker

Description

Directive: w-datepicker

Datepicker widget, allows to select date.

Example

<form role="form">
  <div class="form-group">
    <label>Birth date</label>
    <div w-datepicker="true"
         ng-model="birthDate">
    </div>
    Birth date is: {{ birthDate }}
  </div>
</form>
